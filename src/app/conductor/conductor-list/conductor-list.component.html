Conductores

Como coordinador, para buscar un conductor específico por su nombre, simplemente presione el botón "Buscar". Esta funcionalidad le permitirá acceder de manera rápida y eficiente a la información detallada del conductor que está buscando. Al ingresar el nombre en el campo de búsqueda, obtendrás una lista filtrada de resultados que coincide con el nombre ingresado. Al seleccionar el conductor deseado, podrá ver todos los datos relevantes como su información personal y también editarlos. Esta herramienta está diseñada para facilitar la localización de conductores en la base de datos, asegurando que pueda acceder a la información precisa de manera ágil y sin complicaciones.

También como coordinador, puede evidenciar la información detallada de los conductores de Transmilenio, a través de un listado completo que permite visualizar todos los datos relevantes de cada conductor. Esta tabla ofrece la funcionalidad de editar y actualizar la información de los conductores, asegurando que los datos estén siempre actualizados. Además, tendrás la posibilidad de ver las asignaciones actuales de cada conductor, incluyendo las rutas y horarios que tienen asignados. Esta herramienta está diseñada para facilitar la gestión eficiente del personal, garantizando que la operación diaria del sistema Transmilenio se realice de manera ordenada y precisa.
